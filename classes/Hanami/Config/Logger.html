level(value = nil) Link

Logger #level

Available values are:

* DEBUG
* INFO
* WARN
* ERROR
* FATAL
* UNKNOWN

@overload level(value)

Sets the given value
@param value [Integer,String,Symbol] the logger level

@overload #level

Gets the value
@return [Integer,String,Symbol] returns the level

@since x.x.x

@example Constant (Integer)

require 'hanami/application'

module Bookshelf
  class Application < Hanami::Application
    configure do
      logger.level Hanami::Logger::DEBUG
    end
  end
end

@example String

require 'hanami/application'

module Bookshelf
  class Application < Hanami::Application
    configure do
      logger.level 'debug'
    end
  end
end

@example Symbol

require 'hanami/application'

module Bookshelf
  class Application < Hanami::Application
    configure do
      logger.level :debug
    end
  end
end
